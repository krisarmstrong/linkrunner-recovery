## [1.0.0] - 2024-12-24

### Added
- Initial standalone release (extracted from netally-tools)
- Cross-platform device detection (Windows, Linux, macOS)
- MAC address recovery and update
- Serial number management
- Option bits configuration (802.1x, Reports, Reflector)
- Rotating log file support
- Verbose mode for debugging
- CLI with argparse and validation
